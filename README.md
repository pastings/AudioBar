# AudioBar

> A minimalist, hardware-accelerated desktop audio visualizer widget built with Electron.

---

## Features

* **Real-Time Spectrum Analyzer:** High-density 64-bar frequency visualizer driven by the Web Audio API with ultra-smooth smoothing transitions.
* **Window Drag Lock:** Easily toggle window dragging on or off using the quick-lock action icon to keep your layout pinned.

---
